# leetcode

## Benchmark
```
$ npm run benchmark
$ node benchmark
$ node benchmark -t Roman
```

## Test
```
$ npm test
$ mocha test
$ mocha test -t Roman
```

## Test Java using mocha

$ mocha test -t '001.Two Sum' --java
```

## Test Ruby using mocha

$ mocha test -t '001.Two Sum' --ruby
```
